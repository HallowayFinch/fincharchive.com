# Lexicon
*Internal reference — not for publication*

Use these terms consistently.

| Term | Meaning / Tone | Notes |
|------|----------------|------|
| Interference | Any unexplained overlay of signal/image/pattern; also used as a verb | Core umbrella term |
| Procedure 22 | Recurrent protocol suggesting controlled observation | Parameters intentionally unclear |
| Signal Memory | Hypothesis that material retains information via light/vibration | Use sparingly; avoid “magic” tone |
| RecoveryDaemon | System process that compiles/timestamps/labels recovered material | Can be real software or institutional myth |
| Condition | Archival quality marker (Stable / Degraded / Incomplete / Corrupted) | Appears in headers |
| Provenance | Source medium + chain-of-custody language | Helps realism |
